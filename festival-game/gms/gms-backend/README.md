# GMS Backend

## 가상 환경 설정 방법

### Windows
```
$ pip install virtualenv
```

### Linux
```
$ sudo apt install python-virtualenv
```

가상 환경 설정
```
$ python -m venv <name of environment>
```


가상 환경 실행
```
$ <name of environment>\Scripts\activate
```


가상 환경 실행 후 Flask 설치
```
$ pip install Flask
```
